# teletext\_available #
## Summary ##

`teletext_available` indicates whether teletext is available.

## Output ##

```
<?xml version="1.0" ?>
<command_result>
  ...
  <param name="teletext_available" value="{value}"/>
  ...
</command_result>
```

## Values ##

|               | **Teletext** |
|:--------------|:-------------|
| **Available**   | `1`          |
| **Unavailable** | `0`          |
