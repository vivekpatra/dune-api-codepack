# teletext\_key #
## Summary ##

`teletext_key` indicates a key to send to the teletext interface.

## Input ##

| **Parameter**  | **Value(s)**         |
|:---------------|:---------------------|
| cmd            | set\_playback\_state |
| teletext\_key  | {value}              |
| [[timeout](timeout.md)]  | {n}                  |

## Values ##

|             | **Key**     |
|:------------|:------------|
| **Numeric 0** | KEY\_0      |
| **Numeric 1** | KEY\_1      |
| **Numeric 2** | KEY\_2      |
| **Numeric 3** | KEY\_3      |
| **Numeric 4** | KEY\_4      |
| **Numeric 5** | KEY\_5      |
| **Numeric 6** | KEY\_6      |
| **Numeric 7** | KEY\_7      |
| **Numeric 8** | KEY\_8      |
| **Numeric 9** | KEY\_9      |
| **Up**        | KEY\_UP     |
| **Down**      | KEY\_DOWN   |
| **Clear**     | KEY\_CLEAR  |

## Notes ##

  * Key values are case sensitive.
