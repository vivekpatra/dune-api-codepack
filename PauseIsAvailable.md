# pause\_is\_available #
## Summary ##

`pause_is_available` indicates whether the playback can be paused.

## Output ##

```
<?xml version="1.0" ?>
<command_result>
  ...
  <param name="pause_is_available" value="{value}"/>
  ...
</command_result>
```

## Values ##

|               | **Pause** |
|:--------------|:----------|
| **Available**   | `1`       |
| **Unavailable** | `0`       |
